# Liquidity Lounge Calculator (LLC)

Локальное приложение для управления концентрированной ликвидностью в DeFi.

## 🚀 Быстрый старт (Windows)
1. Убедитесь, что установлен **Python 3.10+**.
2. Запустите `launch.bat` в корне проекта.
3. При первом запуске скрипт создаст виртуальное окружение и установит зависимости.
4. Браузер откроется автоматически на `http://localhost:8501`.

## 📂 Структура проекта
- `app.py` — Главный дашборд.
- `pages/` — Страницы приложения (Ассистент, Добавление, Аналитика, Импорт, Настройки).
- `core/` — Математическое ядро (IL, Волатильность, Рекомендации, Бэктест).
- `data/` — Клиенты API (DexScreener, DeBank, CoinGecko) и БД.
- `models/` — Модели данных (Position).

## 🛠 Основные функции
- **New Position Assistant**: Рекомендации по выбору диапазонов на основе волатильности (Conservative, Balanced, Aggressive).
- **Automated Import**: Загрузка ваших позиций прямо из EVM-кошелька через DeBank.
- **Advanced Recommendations**: Советы по ребалансировке с учетом IL, накопленных комиссий и стоимости газа.
- **Historical Backtest**: Проверка эффективности выбранного диапазона на исторических данных.
