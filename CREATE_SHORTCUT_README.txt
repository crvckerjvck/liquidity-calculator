Чтобы создать ярлык на рабочем столе:
1. Правой кнопкой мыши на рабочем столе -> Создать -> Ярлык.
2. В поле расположения укажи: wscript.exe "C:\путь\к\проекту\launch_silent.vbs"
   (замени путь на реальный, например D:\VibeCode Projects\LiquidityCalculator\launch_silent.vbs)
3. Назови ярлык "Liquidity Lounge Calculator".
4. По желанию смени иконку: свойства ярлыка -> Сменить иконку -> выбрать любой .ico файл.

При первом запуске создастся виртуальное окружение и установятся библиотеки (1-2 минуты). 
После запуска откроется браузер с http://localhost:8501.
